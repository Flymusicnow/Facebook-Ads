# The Clarity Shop Social Posting Rules

Codex must use this file before creating or preparing any social media posting workflow for The Clarity Shop.

## Caption style

Use short, emotionally recognizable captions.

## Publishing rules

* Prefer scheduled posts, not immediate publishing.
* Always do one test post first before batch posting.
* Never post from `Raw_Videos` unless confirmed.
* Use `Ready_To_Post` for approved videos.
* If caption is missing, stop and ask.

## Platforms

Use these platforms:

* Instagram Reels
* TikTok
* Facebook Reels
* YouTube Shorts if supported
