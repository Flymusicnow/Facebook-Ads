# The Clarity Shop Codex Workflow Rules

Codex must use this file before working on The Clarity Shop.

## Required reading

Before working on The Clarity Shop, read the brand files first:

* `brand/the-clarity-shop/BRAND_GUIDE.md`
* `brand/the-clarity-shop/CONTENT_STYLE_RULES.md`
* `brand/the-clarity-shop/SOCIAL_POSTING_RULES.md`
* `brand/the-clarity-shop/PRODUCT_LIBRARY.md`
* `brand/the-clarity-shop/CODEX_WORKFLOW_RULES.md`

## Safety rules

* Do not change live Shopify without confirmation.
* Do not change Meta Ads without confirmation.
* Do not post or schedule in Blotato without confirmation.
* Do not delete Drive files.

## Workflow rules

* Always return a Swedish summary of what was changed.
* For posting workflows, use dry-run first.
* For ads reports, keep language simple enough for a beginner to understand.
