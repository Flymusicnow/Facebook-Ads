# Codex Instructions for Flymusicnow/Facebook-Ads

## The Clarity Shop rule

Before doing any work related to The Clarity Shop, always read:

* brand/the-clarity-shop/BRAND_GUIDE.md

Use that file as the default brand reference for:

* Meta Ads reports
* Blotato captions
* social media schedules
* Shopify copy
* product descriptions
* HTML dashboards
* campaign summaries
* creative ideas

## Safety rules

Do not change live Meta Ads without explicit confirmation.

Do not change live Shopify without explicit confirmation.

Do not post or schedule anything in Blotato without explicit confirmation.

Do not delete Google Drive files.

Do not expose API keys, tokens, secrets, or credentials.

Use dry-run first for posting, scheduling, API, and automation tasks.

## Local to GitHub sync workflow

When working on safe repo files such as:

* documentation
* dashboards
* reports
* generated HTML files
* scripts
* tests
* brand files

Use this default workflow:

1. Make the requested change.
2. Run relevant tests or validation.
3. Show changed files.
4. Commit the change with a clear message.
5. Push the commit to origin/main unless the user explicitly says not to.
6. Confirm the pushed commit hash.
7. Confirm git status is clean after push.
8. If the repo is connected to Vercel, check or clearly report whether deployment verification is available.

Important safety boundary:
Do not automatically execute or push changes involving live external systems without explicit confirmation, including:

* Meta Ads live changes
* Shopify live changes
* Blotato posting, scheduling, editing, or deleting
* Google Drive moving, deleting, uploading, or replacing files
* API token or secret changes

For The Clarity Shop dashboard work:
After dashboard/report changes are tested and committed, push to origin/main by default, unless the user says not to.

## Reporting style

Always return a clear Swedish summary.

Use beginner-friendly language.

Avoid unnecessary technical details unless asked.

For Meta Ads reports, make the result visual, simple, and easy enough for a beginner to understand.

## Brand reminder

The Clarity Shop should feel:

calm, premium, warm, clear, feminine, safe, emotionally intelligent, practical, and easy to understand.

The brand sells clarity, self-respect, emotional relief, decision power, safer relationships, less overthinking, and inner stability.
